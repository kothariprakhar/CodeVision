Data processing pipeline — GitHub URL to structured intelligence
1

Repo Acquisition

Tarball download or git clone. Fetches stars, languages, contributors, commit history.

2

File Manifest

Score & categorize files (entry_point=5, config=4, route=3). Select top 400 non-test files.

3

AST Parsing

Tree-sitter for Python, JS, TS, Go, Java. Regex fallback. Builds dep graph + detects patterns.

4

Doc Parsing

PDF, markdown, image requirements downloaded from Supabase storage and extracted.

5

AI Analysis

4-pass Claude pipeline. 80K tokens/pass. Retry with exponential backoff (max 3x).

6

Risk Assessment

44 tech signatures + static checks + AI-mapped risks. Complexity score 1-10. Cost: $900/day.

7

Business Lenses

Capability graph (8 caps · 3 domains), journey graph, quality report with confidence scores.

8

Storage & Views

6 UI-ready views compiled and stored. Project status set to completed.

OutputBusiness function per module
